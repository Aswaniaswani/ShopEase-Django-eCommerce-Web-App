🛒 ShopEase – Django eCommerce Web App

A full-featured eCommerce web application built using Django, Bootstrap, and JavaScript.
This project supports user authentication, product management, cart system, order processing, and PayPal payment integration.

🚀 Features

👤 User Features
User Registration & Login
Customer Dashboard
Browse Products by Category
Product Details with Reviews
Add to Cart / Buy Now
Checkout System
Order History
Cancel / Return Orders

🛠️ Admin Features
Admin Dashboard
Add / Edit / Delete Products
Manage Orders
Update Order Status
View Users & Analytics

💳 Payment
PayPal Payment Integration
Cash on Delivery (COD)

🛍️ Product Features
Category Filtering
Search Functionality
Price Filtering
Multiple Product Images

🧑‍💻 Tech Stack
Backend: Django (Python)
Frontend: HTML, CSS, Bootstrap 5
Database: SQLite
Payment Gateway: PayPal SDK
